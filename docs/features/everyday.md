毎日0時に `#general` へ次の内容を含むメッセージを送信します。

- 年間通算日数
- 年間残り日数
- 一般社団法人 日本記念日協会「今日の記念日」
- 今日が誕生日のユーザー（[birthday コマンド](../commands/birthday.md) で設定可能）

## 今日の記念日

当日の記念日について、番号付き箇条書きで掲載します。  
記念日の詳細は、記念日の詳細に「毎月」という文字列がない記念日について一つ掲載します。
